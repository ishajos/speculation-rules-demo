# Setup

This demo project uses [Astro](astro.dev) (based on [Astroship template](https://github.com/surjithctly/astroship)) to build and run the development preview 

```bash
npm run dev
# or
yarn dev
# or (recommended)
pnpm dev
```

### Preview & Build

```bash
npm run preview
npm run build
# or
yarn preview
yarn build
# or (recommended)
pnpm preview
pnpm build
```
